# index
E-Modul Pembelajaran Interaktif Geografi Kelas XI ini merupakan media pembelajaran digital berbasis HTML5 yang dirancang untuk mendukung proses pembelajaran Geografi secara mandiri maupun terbimbing. E-modul ini dikembangkan dengan memanfaatkan teknologi flipbook, sehingga memberikan pengalaman belajar yang menarik, interaktif, dan responsif.
